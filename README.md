# fwa-detection-mlops
Healthcare Fraud, Waste &amp; Abuse (FWA) Detection MLOps Pipeline — End-to-end ML platform built on AWS using XGBoost, MLflow, Airflow, FastAPI, BentoML, Docker, SageMaker, and Evidently AI. Targeting the $100B+ Medicare fraud problem using CMS government data with HIPAA-aware design.
